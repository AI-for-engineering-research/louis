# Context: Louis's Portfolio Landing Page

## Purpose
A TypeScript-built portfolio landing page showcasing Louis's work, deployed to GitHub Pages. Will be linked from the class's main landing page as a thumbnail.

## Audience
Primary: Class / academic peers and instructors. The site demonstrates portfolio work and documents AI usage in coursework.

## Key Concepts

### Portfolio showcase
The landing page displays:
- **Bio**: Brief introduction text
- **Projects**: Small project showcases with precomputed animations/diagrams (2 total)
- **Weekly AI Usage Log**: Transparent documentation of AI tool usage during the class, updated weekly (separate /usage page)

The "class context" suggests this is academic coursework with an emphasis on documenting AI assistant usage.

### Deployment
- **URL**: https://AI-for-engineering-research.github.io/louis/ (org GitHub Pages, base path `/louis/`)
- **Build**: Vite + React + TypeScript
- **Assets**: GIFs/diagrams stored in repo
- **Trigger**: Rebuild on push to main

### Projects

#### Project 1: Satellite Imagery vs. Synthetic Contrails
Comparison of real satellite imagery of contrails with synthetic imagery from modeling.
- **Asset**: Animated scene showing flights crossing with satellite imagery updates (precomputed)

#### Project 2: APCEMM Contrail Model
Visualization of the APCEMM contrail simulation model.
- **Assets**: Diagrams showing 2D cross-sections of contrails and simulation mechanics
